ASSETS / IMAGES
===============
The Flipkart Clone uses inline emoji + CSS gradient placeholders
for all images, so no external image files are required.

If you want real photos, place JPG/PNG files here with these names
and update the corresponding emoji divs in loadComponents.js
to use <img> tags instead:

  banner1.jpg   – Mobile Carnival banner
  banner2.jpg   – Fashion Week banner
  banner3.jpg   – Home Makeover banner
  banner4.jpg   – Laptop Sale banner
  banner5.jpg   – Gaming Fest banner
  mobile.jpg    – Category: Mobiles
  laptop.jpg    – Category: Laptops
  shoes.jpg     – Category: Footwear
  fashion.jpg   – Category: Fashion
  ads.jpg       – Advertisement panel
